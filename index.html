Cloud Computing — Pestañas

¿Qué es Cloud Computing?

Cloud Computing (computación en la nube) es la entrega de recursos informáticos (servidores, almacenamiento, bases de datos, redes, software) a través de Internet («la nube») de forma escalable y bajo demanda. Permite acceder a servicios sin necesidad de gestionar la infraestructura física.

¿Qué es SaaS, PaaS e IaaS?

  • SaaS (Software as a Service): aplicaciones listas para usar accesibles por Internet (ejemplo: Gmail, Office 365).
  • PaaS (Platform as a Service): plataformas para desarrollar y desplegar aplicaciones sin gestionar la infraestructura subyacente (ejemplo: Heroku, Google App Engine).
  • IaaS (Infrastructure as a Service): recursos de infraestructura virtualizados (máquinas, redes, almacenamiento) que el usuario configura (ejeplo: AWS EC2, Azure Virtual Machines).

¿Qué es Virtualización?

La virtualización es la creación de una versión virtual (no física) de recursos informáticos como servidores, sistemas operativos, dispositivos de almacenamiento o redes que permite ejecutar múltiples entornos aislados sobre el mismo hardware físico.

¿Qué es Docker?

Docker es una plataforma para crear, distribuir y ejecutar aplicaciones en contenedores. Los contenedores empaquetan la aplicación con sus dependencias, garantizando que se ejecute de forma consistente en cualquier entorno.

Descripcion:

AnyDesk

AnyDesk es una aplicación de acceso remoto y soporte remoto que permite controlar PCs y ofrecer asistencia técnica a distancia mediante una identificación única y conexión segura. Es ligera y optimizada para baja latencia.

TeamViewer

TeamViewer es otra solución popular de acceso y soporte remoto que ofrece conexión remota, transferencia de archivos y reuniones en línea. Funcionan con IDs y contraseñas para sesiones remotas y cifrado de extremo a extremo.
